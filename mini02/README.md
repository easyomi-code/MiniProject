# 📍 MiniProject 02 - Calculator
### ✅ [**02 - Calculator**](https://github.com/easyomi-code/MiniProject/tree/main/mini02)
계산기 만들기
### < 프로젝트 소개 >
사용자가 숫자와 연산자를 클릭하면 계산을 수행하고 결과를 실시간으로 출력하는 간단한 웹 기반 계산기입니다. 
### < 프로젝트 기간 >
2024.07.15(1일)
### < 주요기능 >
1. 사칙 연산
2. 초기화 및 삭제
3. 소수점 입력
4. 실시간 계산
